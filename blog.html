The full meaning of DOM is Document Object Model. It is a structure that shows us the full picture of how objects are classified inside the HTML body. We can easily understand the interface of the code by DOM and change or access it accordingly.

We can select element from DOM by using-
1. querySelectorAll()
2. querySelector()
3. getElementById()
4. getElementByClassName()
5. getElementByTagName()

When an event happens in a particular element in the DOM, it bubbles up and captures all other events of its parent and then their parent, upto the root level. It is called event bubble.

A callback function is a function that is used in another function as an argument. It allows a function to interact with other functions.
